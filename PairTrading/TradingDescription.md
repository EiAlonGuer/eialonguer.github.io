This project uses TIINGO data to test a Trading strategy based on the principle that the assets studied should move together. This site uses a Kalman filter to find a dynamic beta to better represent the underlying relationship.
---
***Live Version***

v1.0 (18/2/26) We're using 10x leverage on the SPY - QQQ spread. No comissions or taxes are taken into account.
